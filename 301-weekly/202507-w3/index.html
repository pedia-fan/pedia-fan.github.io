前言
#

本篇是对 2025-07-14 到 2025-07-20 这周生活的记录与思考。

这周见了高中时期的朋友L，聊了聊各自的近况。我的朋友不多，也很少聊天，平时喜欢一个人待着，有时候会羡慕别人有很多朋友，自己的朋友却总是阶段性的，每次进入人生的新阶段，就会和之前的朋友走散，但自己又很难分配出额外的注意力来维系朋友关系。这件事情曾经困扰着我，所以我想写写现在我对朋友、社交和独处的看法。

朋友关系的全生命周期：开始、维系和结束
#

学生时代的朋友关系，是最容易建立的，尤其是中学时期。学校为大家天然地提供了每天见面的契机，班级又将大家划分为同一个集体，有更多共同的话题，再加上大家年龄尚小，没有太多的利益冲突，交往也更纯粹。所以学生时期的朋友，往往更容易是真心的、靠得住的朋友。再长大一些，会明显感觉到更难交到朋友了，一方面是认识新朋友的契机越来越少，大家都有自己固定的圈子，而结交新朋友的成本太高（要真正认识一个人是一件耗时且耗费精力的事情）；另一方面是成年后价值观基本形成，很难匹配到与自己利益一致、三观相符的朋友。

在形成稳定的朋友关系后，双方对于关系的共同维系也是很重要的。之前我没有认识到这一点，所以很多中学时期的好朋友都在各自进入新的人生阶段后，慢慢地走散了。在缺少共同生活环境的情况下，关系的维系更需要双方的合力，多去主动开启话题，分享生活。

从学校离开后，由于缺少共同的生活环境和话题，大家慢慢失去联系，这也是正常的。基于共同活动和经历而产生的友情，如果没有足够深的交集（指除了共同活动外，对人的深入了解和相处），很可能会在共同活动消失后慢慢瓦解，而这种也大概率会是阶段性的关系。之前总觉得是自己没有好好维护，导致失去了一些朋友，现在更释然一些，接受生活中的绝大部分关系都是阶段性的，而人总是要向前走的，对一些朋友的记忆停留在那些共同经历的美好时光，也挺好的。

对于我而言，由于很难把注意力分配到关系的维系上，因此很多过去的朋友都慢慢不再联系了。一些要好的朋友，虽然平时不会联系，但如果互相需要帮忙，会知道对方是值得信赖的，这样的关系也很好。在前些日子，我还非常幸运地在网络上遇到了一群同频的朋友，我们每天分享生活中的快乐或者难过，接住彼此的情绪，分享自己的挫折或者进步，互相鼓励，变成更好的人。在我困顿迷茫的生活中找到了同路人，感受到建立自己的社会支持体系的重要性，非常感谢我亲爱的朋友们。

保护好自己的社交能量
#

自己经常感觉到社交能量不足，因此观察了一下自己的耗能模式，主要的能量消耗有两点：

  • 我很容易胡思乱想，很在意他人的看法，脑子会自动加工所见所感的各种信息，消耗大量能量；
  • 当聊天内容无法让我产生想法、或者情绪上的波澜时，我的注意力会切换出聊天内容本身，转而关注到其他相处的细节上，进一步产生内耗。

注意力的分配有时候是不自觉的，我想在接下来优化一下信息处理的步骤。当大脑敏锐地察觉到一些信息后，怎样去理解和诠释这些信息。一些信息经过大脑的反复加工，变得极其复杂，比如多步预判他人的想法，变得犹豫不前。这时候要提醒自己，这些想法都是自己人为制造的阻力，是不准确的，甚至是有误差积累的。不要因为假想中的他人评价，而让自己在是否行动的决策上浪费大量精力。直接去行动，是解决问题最高效的方式。

这周的一次实践也印证了这一点，并给了我信心。原本给朋友挑选的礼物临时发现问题，需要更换一个礼物，当我还在纠结朋友有没有CD机的时候，妈妈鼓励我去直接问问对方。我的脑子里立即产生很多预设的阻碍：朋友还不知道我要送她礼物，而且过几天就要见面了，这时候直接开口问对方是不是过于刻意，对方会不会认为我没有用心，如果对方拒绝了怎么办。但是时间紧迫，还是鼓起勇气突破自己，直接询问了朋友。让我意外的是，我收到了朋友真诚、直接的反馈，她没有拒绝或否定我的想法，说自己确实没有，但有可能会放在家里落灰。真实的表达真好啊，完全不会让人有不舒服的感觉，我也感觉被对方很好地接住了。希望自己以后也能少一些纠结和内耗，多一些直接的沟通和真诚的表达。

幸福的小事
#

今晚，让我们重新走进剧场
#

和朋友去看了开心麻花的《出马》，意料之外的好看。相比于我之前喜欢看的严肃话剧，这种喜剧形式的话剧会适合更广泛的人群，不需要专业的鉴赏和理解能力，也不局限于特定年龄段的群体，很适合朋友、家人一起去看。台词很有喜剧的特色，剧情也不逊色，有很多层反转，全程两个半小时都很精彩。我们买的是地质礼堂80元价位的票，视野很好，体验完全值回票价。

拥有一台拍立得，开始认真记录生活的美好
#

收到了朋友送的拍立得，打算用它来记录生活，好好保留下生活中的一些重要时刻。这几年丢失了记录生活的习惯，回看时感觉时间匆匆过去，好像什么都没有留下来。虽然现在用手机拍照清晰且方便，但可能拍立得的意义正在于它的珍贵性。手机拍照过于便捷，以至于我们在拍下海量照片后就不会去回看，有时候也会忘记拍照的目的。而拍立得帮助我们保留下那些足够珍贵的记忆，每次看到照片时都好像回到了那个当下，感受当时的快乐和幸福。
