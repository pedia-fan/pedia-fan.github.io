前言
#

本篇是对 2025-08-04 到 2025-08-10 这周生活的记录与思考。

这周沉住气写完了论文，用Figma制作了简历，开始进入秋招投递阶段。生活上，上周找高中同学玩，度过了短暂但开心的时间，于是这周又找了两个以前的朋友见面、聊天，有几个很真心的朋友，还是挺好的。

这周的话题部分，整理一下船长的直播笔记，关于经济周期的分析和判断，以及个体应该如何把握机会。

康波周期的分析
#

1926年俄国经济学家康德拉季耶夫，发现发达商品经济中存在的一个为期50~60年的经济周期。 在这个周期里，前15年是衰退期；接着20年是大量再投资期，新技术不断采用，经济快速发展；后10年是过度建设期；过度建设的结果是5~10年的混乱期，从而导致下一次大衰退。

——MBA智库百科1

科技进步带来资产重估。本质上，康波周期反映了技术革命、产业结构变迁和经济制度变革带来的长期繁荣与衰退交替。历史案例：1870 蒸汽 → 1880 煤炭 → 钢铁/电力。

对应到现在的年代，我们目前(2025年)正处于衰退期，生活中也可以明显地感受到经济下行，普通打工人和企业家都非常艰难。同时人工智能、大模型引发新一轮的科技革命，2025-2035也是资产重估的时期。再往后，2035-2045/2050，投资泡沫清出，产业结构性调整升级，体感上百废待兴，为下一阶段做铺垫。2050-2060走到应用层，新技术全面商业化，基础设施与配套产业成熟，资本市场进入快速增长期，套利机会多，也是普通人能挣到钱的时机。2060-2070，前期繁荣的技术和资产开始兑现收益，社会进入富足但增长放缓，投机减少，适合躺平，学术研究。至此大致一个周期结束，我们这代人也步入晚年。

通过以上分析可以知道，未来的5-10年经济会逐渐回暖，资产重估完成后，资金会定向流入重点领域，推动真实需求的释放。此外，在大规模商用到来之前，会先出现小范围的商业化应用。一部分活钱也会通过正在布局的精英而流到民间。而在主流玩家/庄家进行资产重组期间，我们要赶在这5-10年的泡沫形成期间去占据先机，具体而言，则取决于我们能够接触什么样的业务？你的岗位离业务是否够近？行业地位如何？

这也是作为刚毕业的大学生，进入职场应该关注的问题。我们可以不在乎职级、职位或者晋升加薪，但要去拿事权，去接近业务与客户，积累履历，建立自己的影响力。我们要在一个领域深耕3-5年，积累行业经验，除了个人工作范畴内的事情，重要的是获得更多企业内部信息，和业内的人聊，理解行业的运行逻辑。同时也要留意在当前平台上的一些机会，比如外派。争取在2035-2045年之前，手上有具体的业务和具体的影响力。

此外，要注意个人影响力的形成。多写技术博客，写自己的沉淀、心路历程、行业分析等，建立个人的影响力。

有创业意向的年轻人，在未来5-10年也不用着急创业。先蛰伏、沉淀，专注于个人的能力提升，无论是专业能力，还是分析力层面，以及重视打造个人的影响力。多去看看外面的人在做什么，不要闭门造车。

在行业方面，尽可能接触前沿行业，以获得（无论哪个国家）政府的补贴和扶持，获取稳定收益。比如人工智能、生物制药（基因工程，脑机，干细胞，3D打印器官），新能源（可控核聚变），区块链（稳定币，即数字主权货币，资产置换的套利空间），robotics（自动化，机器人）。

最重要的是时刻保持思考，保证方向不偏。

幸福的小事
#

终于写完论文了！
#

这个应该可以算一件幸福的事了，在论文这里停滞了一个多月，一直没有推进秋招的进度，很焦虑，现在终于走出来了。一直处于多线并行状态，感觉正在慢慢理顺，一点点完成。

其实真正写论文的时间没有很久，但是整体架构的构思却花了很长时间，以及行动阻力一直比较大，有的地方感觉怎么写都不满意。还是要先去做，不要完美主义。先把事情完成，让自己从焦虑的状态里出来，可能才会有更多好的想法诞生。

见老朋友
#

周末去见了老朋友，说了说话。我通常倾向于约朋友出来聊天，而不是在短暂的相处时间里只作为彼此的玩伴。和一些朋友的相处让我感觉很真实，好像一切都以最自然的形态出现，一些没有被修饰过的、生活原本的质感。这种聊天更像温柔的水面，很平静，不会激起波澜，但温和地包裹着我，让我感到安定与被接纳。像阳光落在水面，闪了一下，又融进了平静里。这确实是幸福的。
